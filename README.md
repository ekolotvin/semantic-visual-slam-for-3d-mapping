## В этом проекте модель семантической сегментации интегрируется в пайплайн Visual SLAM для построения трёхмерной семантически размеченной реконструкции сцены.
Для каждого входного кадра:<br>
1. Предсказывается семантическая маска;<br>
2. Маска интегрируется в SLAM;<br>
3. Строится 3D карта сцены с семантическими метками.

## Обзор проекта
Проект разделён на две основные части:

Часть 1 — Семантическая сегментация<br> 
Обучение и сравнение моделей DeepLabV3+ на дорожных датасетах.

Часть 2 — Интеграция в Visual SLAM<br>
Использование лучшей модели сегментации для построения 3D сцены с семантической разметкой.
